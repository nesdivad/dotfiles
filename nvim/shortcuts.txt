Insert at end of line: G+o
Search and replace: :%s/<old text>/<new text>/g (if you want global)
Visual block mode (windows): Ctrl+q
Change marked text in visual block mode: mark text, press c, and type new text

Move text to new line: :m <number of lines> (-<number> for number of lines above, +<number> for number of lines below)

Indent: '=%', indents scope(?)

# Neotree
Switch between winbar tabs: < (go left), > (go right)

# Macros
- '@o': Insert newline, exit insert mode and go one line up

# Visual mode

- 'vi(': highlight parentheses group: Visual-inner-(
    - Can do with (, [, { and much else 
- 'ci(': change parentheses group: Change-inner-(

# Telescope

- '<Leader>fs': Current buffer fuzzy find
- '<Leader>fg': Live grep (with ripgrep)
    - Install ripgrep: "https://github.com/BurntSushi/ripgrep?tab=readme-ov-file#installation"
- '<Leader>ff': Find files
- '<Leader>fb': Open buffers

# Conform

- Install command line tool for each formatter through Mason or CLI tool.
    - e.g. 'terraform' or 'stylua' from Mason
